# Tornado

A simple Python visualization that generates a tornado-like spiral graphic using NumPy and Matplotlib.

## What It Does

The script plots a spiral scatter plot by computing x/y coordinates from linearly spaced radial and angular values. The result is a funnel-shaped tornado pattern displayed in a borderless plot window.

## Requirements

- Python 3.x
- NumPy
- Matplotlib

Install dependencies if needed:

```
pip install numpy matplotlib
```

## How to Run

```
python tornado.py
```

A window will open displaying the tornado graphic.
